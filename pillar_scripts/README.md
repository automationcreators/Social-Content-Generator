## File Organization

### Naming Convention

**Social Media Scripts:**
```

```

**Topic-Specific Scripts:**
```

```

**Script Variations:**
```

**Summaries:**
```

### Content Structure

Each script file typically includes:
1. **Hook** - Attention-grabbing opening
2. **Body** - Main content with research
3. **CTA** - Call to action
4. **Engagement Prompts** - Questions for comments
5. **Platform Adaptations** - How to use on different platforms
6. **Visual Requirements** - What images/graphics needed

## Best Practices

### When Adding New Scripts

1. **Use date-based filenames** for easy sorting
2. **Create summary documents** for longer scripts
3. **Include research citations** for credibility
4. **Add platform adaptations** for multi-use
5. **Document visual requirements** for designers

### When Organizing Content

1. **Keep latest scripts** in root directory
2. **Archive old scripts** (optional, after 6 months)
3. **Maintain consistent naming** for findability
4. **Update this README** when adding new categories

### When Using Scripts

1. **Check date** to ensure content is current
2. **Review research citations** for accuracy
3. **Adapt for platform** (Twitter, LinkedIn, etc.)
4. **Update statistics** if data has changed
5. **Test hooks** with audience before full rollout
